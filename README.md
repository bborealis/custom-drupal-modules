# _Custom Drupal Modules_
##### _A practice Drupal Site 10/16/2015_

#### By Brian Borealis

## Description

_A practice site with custom Drupal modules._

## Database Setup

* Clone repository
* Start local server
* Go to phpMyAdmin
* Import custom_drupal_modules database from sites/db-backup  
* Add user in privileges:

_user : bborealisAdmin_  
_host : local_  
_password : bborealisPass_  

## Drupal site setup
* Go to localhost:8888  
* Site logins :

_user : bborealis_  
_password : pass_  


## Technologies Used

_This project makes use of:
Drupal 7_
